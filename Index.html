USER MANUAL

1. Install the App

1. Download the APK file.
2. Open the downloaded file.
3. Allow "Install Unknown Apps" if requested.
4. Press Install.

2. Connect BOSS ME-90

1. Turn on your BOSS ME-90.
2. Connect ME-90 to your Android phone using a USB OTG cable.
3. When USB permission appears, press Allow.
4. Open ME90Controller.

3. Using the Controller

• LIVE - Control effects and stomp switches.
• PATCHES - Save and recall your presets.
• EDITOR - Adjust effect parameters.
• SETTINGS - App configuration.

4. Troubleshooting

App cannot connect:
• Check USB OTG cable.
• Reconnect ME-90.
• Accept USB permission.
