USER MANUAL

How to use the app

A quick guide to the main ME90Controller pages.

LIVE Page

Use the live page during practice or performance.

  • Tap an effect to select or control it.
  • Use stomp buttons 1–4 to change patches.
  • Control the WAH section and level display.
  • View bank, patch name and patch number.

EDITOR Page

Use the editor to view and adjust effect parameters.

  • Select an effect from the signal chain.
  • View its available parameters.
  • Adjust type, gain, tone and level values.
  • Changes are sent to the connected ME-90.

PATCHES Page

Use the patches page to store custom setups.

  • Prepare your preferred effect settings.
  • Open the PATCHES page.
  • Press SAVE PATCH.
  • Tap a saved patch later to recall it.

Installation

Install the APK directly on a compatible Android device.

  • Press the Download APK button.
  • Open the downloaded APK file.
  • Allow installation from this browser if asked.
  • Press Install and wait for completion.

HELP

Troubleshooting

The app does not detect my ME-90

Check that your phone supports USB OTG. Reconnect the cable, turn on the ME-90 and accept the USB permission request.

I pressed Download but the APK did not install

Open your phone's Downloads folder and tap me90controller.apk. Android may ask you to allow installation from your browser or file manager.

The page does not show the screenshots

Confirm that live.png, editor.png and patches.png are in the same GitHub repository folder as boss.html. File names must match exactly.

The download button shows file not found

Confirm that the APK is named exactly me90controller.apk and is located beside boss.html.
